### 一、相关技术
- 前端：`jsp`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1. 登录、2. 注册、3. 首页、4. 全部服务、5. 服务详情、6. 立即预约、7. 网站公告、8. 个人中心、9. 我的订单

管理员功能：1. 登录、2. 新增管理员信息、3. 管理员信息列表、4. 用户信息管理、5. 新增新闻公告、6. 新闻公告管理、7. 新增服务类型、8. 服务类型管理、9. 新增服务项目、10. 服务项目管理、11. 订单信息管理、12. 订单项信息管理、13. 订单评价管理、14. 新增工作人员、15. 工作人员管理、16. 订单分配、17. 订单分派管理、18. 数据统计
